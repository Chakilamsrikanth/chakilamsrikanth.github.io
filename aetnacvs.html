Aetna & CVS Health

In Aetna and CVS implemented a web-based UI application using HTML, CSS, JavaScript, jQuery, BackboneJS, and AJAX. Proficient in leveraging Adobe tools for graphic design, securing RESTful APIs with OAuth2, and adept at utilizing Java Spring Boot for crafting efficient server-side logic, resulting in an enhanced user experience and a remarkable 40% reduction in database response times. The collaborative projects undertaken by Aetna and CVS Health have focused on developing sophisticated web applications, seamlessly integrating both frontend and backend technologies to enhance user interaction and ensure robust system functionality.

     

Aetna's front-end implementation embraces visually appealing login pages and user dashboards, providing an intuitive and seamless user experience. The backend infrastructure relies on Java, Spring Boot, Hibernate, and Apache Maven, forming a powerful foundation for data management, business logic implementation, and overall backend application structure. both RESTful and SOAP APIs are integrated into the system. Aetna's project is hosted on AWS (Amazon Web Services), leveraging the scalability, reliability, and security features of cloud technology.
