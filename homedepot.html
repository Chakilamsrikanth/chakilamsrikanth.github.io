The Home Depot

At The Home Depot, I spearheaded projects aimed at captivating crafting consumer experiences through a blend of design prowess and technical finesse. Leveraging a deep understanding of design principles and user behavior, I led the development of Spring Boot frameworks with RESTful APIs, delivering seamless user interfaces utilizing HTML5, CSS3, Bootstrap, jQuery, and Angular JS. I meticulously monitored performance using tools like New Relic and Splunk, swiftly addressing any issues that arose. To streamline our development process, I implemented and managed Jenkins CI tool on AWS instances, ensuring efficient continuous integration. Furthermore, I integrated data from Big Query and MySQL databases, creating comprehensive views using Spring Boot. These views were then consolidated to generate essential front-end tables, such as purchase orders and shipment details, utilizing Angular. By orchestrating dynamic web experiences and optimizing performance and scalability through backend expertise in Java and GCP data migration, I played a pivotal role in enhancing The Home Depot's digital ecosystem.
